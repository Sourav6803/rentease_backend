# RentEase API Documentation

## 📚 Overview

Welcome to the RentEase API documentation. This API powers the furniture and appliance rental platform, providing endpoints for user management, product catalog, rentals, payments, and more.

## 🚀 Quick Start

### Base URL

Production: https://api.rentease.com/api/v1
Staging: https://staging-api.rentease.com/api/v1
Development: http://localhost:5000/api/v1
